# Data_logger Documentation

Knowledge base for the Data_logger project (ScanImage data-structure and
project logging automation). Generated by structural survey; update as the
codebase changes.

- **[ARCHITECTURE.md](ARCHITECTURE.md)** — project purpose, components,
  data flow, how to run, dependencies, NWB status.
- **[FUNCTIONS.md](FUNCTIONS.md)** — per-file function reference (signature
  + description) for every `.m` file in the repo, grouped by directory.
- **[KNOWN_ISSUES.md](KNOWN_ISSUES.md)** — duplicate implementations, a
  function-name collision, a likely copy-paste bug, a missing dependency,
  and dev/exploratory code to be aware of before extending the pipeline.

For installation and quick-start instructions, see the
[root README](../README.md).
